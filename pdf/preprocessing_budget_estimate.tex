\DocNeedspace{10\baselineskip}
\section*{5. LLM 태깅 비용 주의사항}
\addcontentsline{toc}{section}{5. LLM 태깅 비용 주의사항}


\DocNeedspace{3\baselineskip}
LLM 태깅은 모델 선택에 따라 비용 차이가 크다.


\DocNeedspace{3\baselineskip}
저가 태깅 모델을 사용하면 20,000건 기준 LLM 비용은 약 \texttt{\$2.\allowbreak{}76}, 원화 약 4,179원 수준으로 예상한다.


\DocNeedspace{3\baselineskip}
반면 Claude Sonnet급 모델로 전건 태깅을 수행하면 동일한 20,000건 기준으로 LLM 비용만 약 \texttt{\$168}, 원화 약 254,352원 수준까지 증가할 수 있다.


\DocNeedspace{3\baselineskip}
따라서 PoC 기본 정책은 다음과 같이 둔다.


\begin{itemize}
\item 규칙 기반 태깅을 1차로 수행한다.
\item 카테고리·설명·국가별 표현 차이 때문에 애매한 데이터만 LLM으로 보강한다.
\item LLM 결과에는 \texttt{tag\_\allowbreak{}source}, \texttt{tag\_\allowbreak{}confidence}, \texttt{tag\_\allowbreak{}status}를 남긴다.
\item 운영 전환 시 Lambda 내부 직접 호출보다 비동기 태깅 Job으로 분리한다.
\end{itemize}
