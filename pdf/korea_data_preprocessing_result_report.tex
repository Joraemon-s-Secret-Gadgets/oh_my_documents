\subsection*{2. 작업 목적}

\DocNeedspace{3\baselineskip}

\subsection*{4. 전처리 방식}
\addcontentsline{toc}{subsection}{4. 전처리 방식}


\DocNeedspace{3\baselineskip}
전처리는 \texttt{kr-\allowbreak{}domain-\allowbreak{}loader} Lambda에서 수행했다.\newline Lambda는 S3 Raw JSON을 읽고, Python 전처리 모듈을 통해 도메인별 item을 생성한 뒤 DynamoDB에 저장한다.


\DocNeedspace{3\baselineskip}
처리 흐름은 다음과 같다.


\DocNeedspace{7\baselineskip}
\begin{quote}
\begin{scriptsize}
\ttfamily
\RaggedRight
\setlength{\parskip}{1pt}
\textcolor{LovvGreen}{\textbf{text}}\\[0.2em]
S3 Raw JSON\\
↓\\
kr-\allowbreak{}domain-\allowbreak{}loader Lambda {\normalfont 실행}\\
↓\\
{\normalfont 도시} {\normalfont 메타데이터} {\normalfont 생성}\\
↓\\
Raw content {\normalfont 순회}\\
↓\\
{\normalfont 음식점} /\allowbreak{} {\normalfont 관광지} /\allowbreak{} {\normalfont 축제} /\allowbreak{} {\normalfont 방문} {\normalfont 통계} {\normalfont 분리}\\
↓\\
{\normalfont 도메인별} {\normalfont 컬럼} projection\\
↓\\
Dynamo\allowbreak{}DB PK/\allowbreak{}SK {\normalfont 및} GSI {\normalfont 속성} {\normalfont 생성}\\
↓\\
Tour\allowbreak{}Korea\allowbreak{}Domain\allowbreak{}Data {\normalfont 적재}\\
↓\\
processed/\allowbreak{}KR/\allowbreak{}domain summary {\normalfont 기록}\\
\end{scriptsize}
\end{quote}
