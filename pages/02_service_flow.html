1. 문서 개요

1.1 목적

본 문서는 로브 서비스의 사용자 진입, 온보딩, 지도 탐색, 챗봇 추천, 결과 확인, 저장·피드백 흐름을 화면과 상태 전이 관점에서 정의한다. 요구사항 명세서의 기능 요구사항을 화면 흐름과 사용자 행동 단위로 연결해 UX 설계와 프론트엔드 구현의 기준으로 사용한다.

1.2 적용 범위

  • 신규 사용자 필수 온보딩
  • 메인 지도 기반 소도시 탐색
  • 지도 마커 클릭 추천 진입
  • 챗봇 직접 질의 추천 진입
  • 축제·행사 포함 여부 확인
  • 소도시 1곳 집중 일정 추천 결과
  • 일정 저장, 좋아요/싫어요 피드백, 마이페이지 조회

2. 사용자 유형과 진입점

사용자진입점핵심 목적
신규 자유여행자온보딩 → 메인 지도 또는 챗봇선호 테마를 설정하고 첫 추천을 받는다
재방문 자유여행자메인 지도 또는 챗봇저장 선호와 피드백을 반영한 추천을 받는다
관광 운영자운영 화면담당 지역 데이터와 제안 상태를 확인한다
서비스 관리자관리자 화면데이터 제안 승인, 추천 정책, 공지사항을 관리한다

3. 전체 서비스 흐름

단계화면/상태사용자 행동시스템 처리다음 상태
1서비스 진입앱 또는 웹 첫 화면 접근온보딩 완료 여부 확인온보딩 또는 메인
2온보딩대도시 스타일 선택선택지를 6개 기본 테마로 매핑메인 지도
3메인 지도국가·월·테마 필터 조정소도시 마커 목록 필터링지도 탐색
4A지도 마커 팝업소도시 마커 클릭목적지 고정일정 유형 선택
4B챗봇자연어 조건 입력조건 구조화 및 진입점 분류축제 포함 여부 확인
5추천 직전 확인축제·행사 포함 여부 선택세션 추천 조건에 반영추천 생성
6추천 결과일정 카드 확인소도시 1곳 일별 세부 일정 표시저장·피드백
7상세 패널지도, 현재 날씨, 외부 링크 확인외부 API/딥링크 정보 표시결과 화면 유지
8저장·피드백저장, 좋아요/싫어요 선택로컬 스토리지 또는 계정 저장마이페이지/다음 추천

4. 온보딩 흐름

4.1 조건

  • 온보딩은 신규 사용자에게 필수로 제공한다.
  • PoC에서는 브라우저 로컬 스토리지에 완료 여부와 선호 프로필을 저장한다.
  • Production에서는 로그인 사용자 계정에 선호 프로필을 저장한다.

4.2 질문 구조

단계질문선택 결과
1어떤 스타일의 여행지를 선호하나요?한국 또는 일본 대도시 스타일 선택
2선택 확인대도시 스타일을 기본 테마로 매핑
3완료선호 테마 프로필 저장

4.3 대도시 스타일 매핑

5. 지도 진입 흐름

5.1 메인 지도 상태

메인 지도는 추천 대상 소도시를 마커로 표시한다. 사용자는 국가, 월, 테마 필터를 조합해 마커를 좁힐 수 있다.

UI 요소동작
국가 필터한국, 일본, 전체 중 선택
월 필터1월~12월 중 선택
테마 필터기본 테마와 확장 테마 선택
소도시 마커클릭 시 목적지 후보로 고정

5.2 마커 클릭 후 흐름

단계사용자 행동시스템 처리
1마커 클릭소도시명, 대표 태그, 추천 월 표시
2일정 유형 선택당일치기~4박5일 중 하나를 추천 조건에 추가
3축제·행사 포함 여부 선택세션 조건에만 저장
4추천 시작해당 소도시 1곳의 일별 세부 일정 생성

6. 챗봇 진입 흐름

6.1 자연어 입력 예시

입력 예시구조화 조건
"6월에 비 덜 오는 일본 소도시 추천해줘"국가=일본, 월=6월, 진입점=월별 기상 경향
"혼자 조용히 쉬는 바다 여행 가고 싶어"동행=혼자, 테마=바다·해안/휴양
"가을에 축제 있는 한국 소도시 알려줘"국가=한국, 시기=가을, 축제 포함 후보

6.2 챗봇 추천 생성 전 필수 확인

챗봇은 추천 생성 직전 아래 항목을 확인한다.

확인 항목기본 처리
국가명시되지 않으면 한국/일본 중 선택 질문
여행 시기월 또는 계절 입력 요청
일정 유형명시되지 않으면 당일치기~4박5일 선택 질문
축제·행사 포함 여부예/아니오 선택 질문

7. 추천 결과 흐름

7.1 결과 카드 구성

영역표시 내용
추천 대상소도시명, 국가, 지역, 대표 이미지
추천 이유사용자 조건, 계절 적합도, 혼잡도, 월별 기상 경향 중 2개 이상
일정일정 유형에 맞춘 일별 방문 순서
축제·행사사용자가 포함을 선택한 경우 해당 월 행사 정보
외부 링크지도, 공식 관광 정보, 숙소 검색 링크, 현지 플랫폼 링크
피드백좋아요, 싫어요, 저장

7.2 대체 일정 토글

월별 기상 경향상 우천, 폭설, 장마, 태풍 가능성이 높은 목적지는 "기상 악화 시 대체 일정 보기" 토글을 제공한다. 토글을 활성화하면 동일 목적지 안에서 실내 중심 일정으로 전환한다.
