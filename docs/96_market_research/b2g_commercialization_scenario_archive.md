> 해석 주의: 이 파일은 LOVV를 영리화할 경우를 전제로 한 가정형 사업 구상 복원본이다. 아래 가격·방문·소비 수치는 사업성 검증을 위한 내부 가설이며 확정 판매가격, 계약조건 또는 보장 성과가 아니다. 비영리·혼합형을 포함한 현재 공공협력 검토는 `b2g_public_sector_cost_effectiveness_research.md`를 기준으로 한다.

본 보고서는 `LOVV를 영리 서비스로 운영한다면 어떤 고객에게 무엇을 판매하고, 어떤 성과로 재계약과 확장을 만들 것인가`를 검토한다.
초기 시장은 지자체·관광공사·관광재단 등 공공관광 발주기관으로 설정하고, 지역 데이터 온보딩·추천 플랫폼 이용·테마 캠페인 운영·성과분석을 결합한 B2G 상품을 핵심 매출원으로 가정한다.

충주시 유튜브와 예산상설시장은 그 자체를 평가하기 위한 대상이 아니다. 두 사례를 벤치마크로 사용해 LOVV가 판매할 상품, 고객이 구매할 이유, 예산 대비 검증할 성과를 제시한다.

LOVV는 공공기관이 보유한 관광자원과 홍보 콘텐츠를 `테마 추천 → 일정 저장 → 지도·외부 링크 이동 → 방문 확인`으로 연결하고, 어느 테마와 지역이 실제 행동을 만들었는지 측정하는 추천·성과분석 플랫폼으로 가정한다.

영리화의 기본 논리는 `지역 단위 PoC 수주 → 성과 검증 → 연간 라이선스·운영 계약 전환 → 다지역·다테마 확장`이다. 사용자 추천의 공정성과 데이터 신뢰성은 유료 계약 여부와 분리하며, 기관의 비용 지불이 추천 순위를 직접 결정하지 않도록 한다.

# 1. 결론 요약

## 1.0 영리화 사업 가설

| 항목 | 가정 |
| --- | --- |
| 초기 고객 | 시·군·구 관광부서, 지역관광공사·관광재단·DMO, 전통시장·축제 사업단 |
| 판매 상품 | 지역 데이터 구축, 테마 추천·일정 플랫폼, 캠페인 운영, 성과 대시보드·보고 |
| 매출 구조 | 초기 구축비 + 기간형 플랫폼 라이선스 + 데이터 운영비 + 캠페인·분석 과금 |
| 진입 전략 | 6개월 지역 PoC로 전환성과와 운영 가능성을 검증한 뒤 연간 계약으로 전환 |
| 확장 전략 | 동일 지역의 테마 확대, 인접 시군 추가, 광역권·다지역 라이선스로 확장 |
| 핵심 사업지표 | 계약 전환율, PoC의 연간 계약 전환율, 연간 반복매출, 지역·테마당 운영원가, 재계약률 |

이 구상에서 공공기관이 지불하는 비용은 추천 순위 구매비가 아니라 데이터 구축·운영, 추천 기능 사용, 캠페인 실행, 성과측정에 대한 대가다.

## 1.1 LOVV가 관공서에 제공하는 가치

| 공공사업의 기존 단절 | LOVV 적용 | 기대효과 |
| --- | --- | --- |
| 홍보 콘텐츠가 조회수에서 끝남 | 영상·SNS·보도자료를 테마형 소도시 추천과 일정으로 연결 | 콘텐츠 관심을 여행 계획과 방문 의도로 전환 |
| 유명 관광지 중심 노출 | 계절·취향을 반영한 소도시 추천과 주변 상권 노출(주: 대도시 입력에 따른 능동 대체 추천은 현재 미구현이며, `96_market_research.md` 기준 포지셔닝 메시지로만 사용) | 방문 수요 분산과 비인기 자원 노출 확대 |
| 관광 데이터가 기관별로 분산 | 지자체 제안, 출처, 갱신일, 검증 상태를 추천 데이터로 관리 | 공공 데이터의 실제 활용 경로 확보 |
| 성과가 조회수·노출 수에 머묾 | 노출, 클릭, 저장, 길찾기, 방문 설문을 같은 캠페인 ID로 연결 | 예산 대비 방문 전환 성과 측정 |

LOVV의 핵심 상품은 AI 여행 추천 자체보다 공공 관광자원을 테마로 재구성하고 추천 이후 행동을 측정하는 운영 체계로 정의했다.

## 1.2 예산안별 핵심 가정

아래 금액은 확정 판매가격이나 시장 평균이 아니라, 영리화 시 공공기관 제안서와 PoC 협의에 사용할 LOVV 내부 가격·성과 가설이다.

| 구분 | 제안 예산 | 최소 성과 기준 | 목표 성과 기준 | 예산 대비 해석 |
| --- | ---: | ---: | ---: | --- |
| 6개월 지역 PoC | 5,000만 원 | 측정 가능한 추가 방문 5,000건 | 10,000건 | 방문당 1만 원 이하, 목표 달성 시 5,000원/건 |
| 1년 지역 운영 | 1억 원 | 측정 가능한 추가 방문 10,000건 | 20,000건 | 방문당 1만 원 이하, 목표 달성 시 5,000원/건 |
| 1년 확장 목표 | 1억 원 | - | 50,000건 | 방문당 2,000원/건 |

`방문당 1만 원`은 충주시가 단체관광객 유치 여행사에 지급한 인센티브의 공개 하단값인 1인당 1만 원을 비교 기준으로 사용했다.
LOVV 방문은 인센티브 조건을 충족한 단체관광객과 동일하지 않으므로 사업 타당성 검토를 위한 성과 기준으로만 사용한다.

방문객 1인당 지역 내 소비를 7만 원으로 가정하면 다음과 같다.

| 운영안 | 추가 방문 | 지역소비 유발액 가정 | 사업예산 대비 총소비 배수 |
| --- | ---: | ---: | ---: |
| 5,000만 원 PoC 최소 기준 | 5,000건 | 3억 5,000만 원 | 7배 |
| 5,000만 원 PoC 목표 기준 | 10,000건 | 7억 원 | 14배 |
| 1억 원 연간 운영 목표 | 20,000건 | 14억 원 | 14배 |
| 1억 원 연간 운영 확장 목표 | 50,000건 | 35억 원 | 35배 |

이는 세수, 순이익 또는 LOVV 매출 ROI가 아니라 `추가 방문객 수 × 1인당 소비 가정`으로 계산한 지역 내 총소비 유발액이다.

## 2.1 현재 기능의 공공사업 활용 가정

| LOVV 기능 | 공공기관 활용 | 측정 지표 |

| RAG 기반 근거 설명 | 지자체·관광공사 데이터의 출처와 추천 이유 표시 | 출처 열람률, 추천 신뢰도 |

| 운영 지표 대시보드 | 지역·테마·기간별 성과 비교 | 노출→저장→방문 단계별 전환율 |

## 2.2 테마 추천의 기대효과

| 테마 적용 예시 | 추천 구성 | 기대효과 |
| --- | --- | --- |
| 자연·트레킹 + 혼잡 회피 | 유명 관광지 대신 호수길, 숲길, 지역 체험을 1일 일정으로 구성 | 비인기 지점 노출, 체류시간 증가, 수요 분산 |
| 미식·노포 + 역사·전통 | 전통시장, 로컬 음식, 인근 문화자원을 반일·1일 일정으로 구성 | 시장 방문을 주변 상권 소비로 확장 |
| 축제 + 월별 기상 경향 | 행사 일정과 계절 적합 장소, 우천 대안을 함께 추천 | 축제 방문 의사결정 지원, 취소 위험 완화 |
| 미디어 성지 + 예술·감성 | 촬영지·작품 연관 장소와 로컬 전시·카페를 연결 | 특정 팬층 유입, 희소 테마의 목적 방문 창출 |

기대효과는 다음 전환 사슬로 측정한다.

`테마 추천 노출 → 목적지 클릭 → 일정 생성·저장 → 지도·길찾기 이동 → 방문 인증 → 지역소비`

# 4. 두 사례를 LOVV 효과 산정에 사용하는 방법

## 4.1 사례 1: 충주시 유튜브

내부 인력 완전원가를 6,000만 원으로 가정하면 연간비용은 6,061만 원이다.
2023년 순증 방문 약 168만 명 중 충TV 기여율을 5%로 가정하면 귀속 방문은 약 8.4만 명이고 방문 증가 1건당 비용은 약 723원이다.

`60,610,000원 / (1,677,559명 × 5%) = 약 723원/건`

이는 충TV의 확정 성과가 아니라 기여율을 가정한 민감도 계산이다.
LOVV 적용 시에는 콘텐츠별 추천 링크, QR, 일정 저장, 지도 이동과 방문 설문을 연결해 실제 기여율을 식별하는 구조를 검토했다.

5. 가능하면 카드매출·통신 방문 데이터를 월·상권 단위로 결합한다.

# 7. 시장성 및 사업화 판단

## 7.1 시장성이 있다고 판단한 근거

1. 공공기관은 유튜브·SNS·관광 활성화에 수천만 원에서 수억 원 단위의 사업비를 집행한다.
2. 조회·노출과 실제 방문·소비를 같은 캠페인 단위로 연결한 공개 성과자료는 부족하다.
3. LOVV의 테마 추천, 소도시 일정, 출처 관리, 지자체 데이터 제안 기능은 공공 관광사업 요구와 연결될 수 있다.
4. 한 지역에서 만든 추천·측정 구조를 다른 시군과 테마로 반복 적용할 수 있다고 가정했다.
5. 혼잡 분산, 비수기 활성화, 전통시장·축제·소도시 연계 등 정책 목적별 제안이 가능하다.

## 7.3 영리화 가정의 최종 판단

LOVV의 B2G 시장성은 기술 소개만으로 성립하지 않으며 정책 목적, 사용자 효용, 성과 증명을 동시에 충족해야 한다고 판단한다.
초기 제안은 5,000만 원·6개월 PoC로 `추가 방문 5,000건 이상 또는 방문당 비용 1만 원 이하`를 검증하고, 기준 충족 시 1억 원 규모 연간 운영으로 전환하는 단계형 계약을 가정했다.

따라서 영리화 초기에는 대규모 플랫폼 라이선스를 바로 판매하기보다 측정 가능한 단일 지역·테마 PoC를 수주하고, 성과와 운영원가를 확인한 뒤 반복매출 계약으로 전환하는 전략이 적합하다. 사업 지속 여부는 방문·소비 목표뿐 아니라 PoC 수주율, 연간 계약 전환율, 지역당 데이터 운영원가, 매출총이익률, 재계약률을 함께 확인해 판단한다.

이 결론은 LOVV 영리화를 전제로 한 사업 시나리오이며 실제 사업방향과 가격은 별도 의사결정과 고객 검증을 거쳐 확정한다.
