# LOVV 비영리 B2G 공익 협력 및 월간 여행지 노출 운영안

> 문서 버전: v0.8
> 문서 상태: Review
> 작성일: 2026-06-18
> 사업 성격: 비영리 공익형
> 문서 목적: 지자체·공공기관과 함께 지역 관광자원의 발견과 방문을 돕는 월간 노출 운영 방향 정의
> 핵심 흐름: 이번 달 여행지 노출 → 상세 클릭 → 일정 탐색 → 공식 정보 이동 → 방문 의향·방문 확인
> 제외 범위: 예산, 과금, 수익, 계약금액, 투자 대비 효과 등 재정적 사유와 경제효과 산정

본 문서는 LOVV의 B2G 방향을 지자체·관광공사·관광재단·공공기관과의 **비영리 공익 협력**으로 정의한다.
기관은 검증된 관광지·축제·지역문화 정보를 제공하고, LOVV는 이를 사용자 취향과 여행 시기에 맞는 `이번 달 여행지` 콘텐츠로 노출한다.

핵심 목적은 특정 지역을 광고 상품처럼 판매하는 것이 아니라, 상대적으로 덜 알려진 지역에 공정한 발견 기회를 제공하고 사용자의 클릭과 여행 계획 행동을 실제 방문 가능성으로 연결하는 것이다.
성과는 재정 수치가 아니라 노출, 클릭, 일정 생성·저장, 공식 정보 이동, 방문 의향과 확인 가능한 방문 반응으로 측정한다.

# 1. 문서 범위와 운영 원칙

## 1.1 이 문서에서 정하는 것

- 비영리 B2G 공익 협력의 목적과 참여 주체의 역할
- `이번 달 여행지` 선정·노출·클릭 유도 방식
- 클릭 이후 일정 탐색과 공식 정보 이동을 연결하는 사용자 흐름
- 지역 관광자원의 발견과 방문효과를 확인하는 비재정 성과지표
- 공공 데이터 제공, 검수, 게시, 성과 공유와 갱신 절차
- 추천 공정성, 개인정보 보호와 공공성 기준

## 1.2 이 문서에서 제외하는 것

| 제외 항목 | 처리 원칙 |
| --- | --- |
| 예산·운영비·계약금액 | 본 문서에서 다루지 않음 |
| 과금·수익·수익 배분 | 비영리 공익 협력의 성과 기준에서 제외 |
| 투자 대비 효과·지역매출·세수 | 산정하거나 성과로 제시하지 않음 |
| 유료 노출·추천 순위 구매 | 허용하지 않음 |
| 방문자 수 보장 | 노출이나 클릭만으로 실제 방문을 단정하지 않음 |

본 문서에서 B2G는 정부·지자체를 구매자로 보는 영업 모델이 아니라, 공공기관과 공식 정보를 연계하고 지역 관광자원의 발견을 돕는 협력 관계를 의미한다.

# 2. 비영리 B2G 협력 구조

## 2.1 참여 주체별 역할

| 주체 | 제공하는 것 | 얻는 공익적 가치 |
| --- | --- | --- |
| 여행 사용자 | 테마 선택, 상세 클릭, 일정 생성·저장, 선택적 방문 피드백 | 취향과 시기에 맞는 신뢰 가능한 소도시 발견 |
| 지자체·공공기관 | 공식 관광정보, 축제·행사 일정, 출처, 변경·휴장 정보 | 지역 관광자원 노출, 공식 정보 유입, 관심·방문 반응 확인 |
| 지역 관광 운영자 | 체험·문화·시설의 최신 운영정보 | 적합한 여행자에게 발견될 기회 |
| LOVV | 월간 큐레이션, 추천 근거, 일정 구성, 공식 링크, 익명 성과 집계 | 추천 품질 향상, 검증 데이터 확보, 사용자 신뢰 강화 |

협력의 기본 흐름은 다음과 같다.

`공식 정보 제공 → LOVV 검수·테마 매핑 → 이번 달 여행지 노출 → 사용자 클릭·일정 탐색 → 공식 정보 이동·방문 반응 → 익명 성과 공유`

## 2.2 협력의 공공성

- 기관 협력 여부와 관계없이 동일한 정보 품질 기준을 적용한다.
- 협력기관이 추천 순위나 노출 문구를 임의로 결정할 수 없다.
- 유료 노출, 순위 구매, 경쟁 지역 배제 방식은 사용하지 않는다.
- 지역 규모와 기존 인지도보다 계절 적합성, 정보 신뢰도, 혼잡 분산 가능성과 사용자 효용을 우선한다.
- 사용자가 왜 해당 지역을 이번 달 여행지로 보게 됐는지 추천 근거를 확인할 수 있게 한다.

# 3. `이번 달 여행지` 노출 모델

## 3.1 운영 목적

`이번 달 여행지`는 사용자가 여행지를 정하기 전 단계에서 계절과 테마에 맞는 소도시를 발견하도록 돕는 월간 공익 큐레이션이다.
매월 지역 관광자원을 반복적으로 소개하되, 단순 배너 노출에서 끝내지 않고 상세 클릭과 일정 탐색으로 이어지도록 구성한다.

노출의 목적은 다음과 같다.

- 검색량이 적은 지역에 새로운 발견 경로 제공
- 특정 유명 관광지에 집중된 관심을 주변 지역과 대체 여행지로 분산
- 축제·계절 명소·전통시장·문화체험 등 시의성 있는 공식 정보 전달
- 사용자의 클릭을 일정 생성, 지도 확인과 공식 정보 탐색으로 연결
- 방문 의향과 선택적 방문 확인을 통해 지역 콘텐츠의 실질 반응 파악

## 3.2 선정 기준

| 기준 | 확인 내용 |
| --- | --- |
| 월별 적합성 | 해당 월의 기상 경향, 개화·단풍·설경, 축제·행사 시기와의 적합성 |
| 테마 적합성 | 자연·트레킹, 미식·노포, 역사·전통, 축제, 예술·감성 등 사용자 테마와의 관련성 |
| 정보 신뢰도 | 공식 출처, 갱신일, 운영 여부와 유효기간 확인 가능성 |
| 혼잡 분산 | 유명 관광지와 유사한 경험을 제공하면서 상대적으로 덜 붐비는 대안인지 여부 |
| 일정 구성성 | 단일 장소가 아니라 반일·1일 이상의 여행 동선으로 연결 가능한지 여부 |
| 접근 정보 | 교통, 운영시간, 휴장, 예약 등 방문 결정에 필요한 정보 제공 가능성 |

선정 지역은 매월 고정 순위를 부여하지 않는다.
사용자의 국가, 여행 월, 선호 테마, 일정 유형과 혼잡 회피 선호에 따라 같은 달에도 노출 결과가 달라질 수 있다.

## 3.3 노출 카드 구성

`이번 달 여행지` 카드에는 다음 정보를 우선 노출한다.

- 지역명과 대표 이미지
- 이번 달에 추천하는 이유
- 대표 테마와 예상 여행 분위기
- 공식 확인이 끝난 축제·행사·계절 정보
- 혼잡한 유명 여행지와 구분되는 대체 경험
- `자세히 보기`, `일정 만들기`, `공식 정보 확인` 클릭 동선
- 기관명, 정보 출처와 최종 갱신일

과장된 방문효과, 경제효과 또는 검증되지 않은 인기 수치는 표시하지 않는다.

## 3.4 사용자 전환 흐름

| 단계 | 사용자 행동 | LOVV 제공 내용 | 기대 효과 |
| --- | --- | --- | --- |
| 1. 노출 | 홈·지도·추천 화면에서 이번 달 여행지 확인 | 월·테마·지역명이 포함된 큐레이션 카드 | 지역 인지와 발견 기회 확대 |
| 2. 클릭 | `자세히 보기` 선택 | 추천 이유, 장소, 축제, 출처와 지도 정보 | 지역에 대한 구체적 관심 형성 |
| 3. 탐색 | 장소·테마·동선 확인 | 반일·1일 일정과 주변 관광자원 연결 | 단일 관심을 여행 계획으로 확장 |
| 4. 계획 | 일정 생성·저장 또는 지도 열기 | 이동 동선, 운영정보와 공식 링크 제공 | 방문 준비 행동 강화 |
| 5. 이동 | 교통·축제·기관 페이지 클릭 | 최신 공식 정보로 연결 | 실제 방문에 필요한 정보 접근 |
| 6. 확인 | 선택적으로 방문 여부·만족도 응답 | 개인정보를 최소화한 피드백 수집 | 방문 반응과 콘텐츠 개선점 확인 |

# 4. 기대하는 공익 효과

## 4.1 지역 관광자원 발견 확대

사용자가 유명 도시명을 직접 검색하지 않아도 월과 테마를 통해 소도시와 지역자원을 발견할 수 있다.
이는 기존 검색과 대형 플랫폼에서 노출되기 어려운 장소에 새로운 발견 경로를 제공한다.

## 4.2 클릭을 통한 방문 가능성 확대

노출만으로는 여행 행동이 만들어지기 어렵다.
LOVV는 `자세히 보기`, `일정 만들기`, `지도 열기`, `공식 정보 확인`을 연결해 관심이 구체적인 계획 행동으로 이어지게 한다.

클릭은 실제 방문과 동일하지 않지만, 상세 탐색과 일정 저장, 교통·공식 정보 이동이 함께 발생하면 방문 의향이 강화된 것으로 볼 수 있다.
선택적 사후 피드백이나 기관이 제공하는 비식별 방문 통계로 확인 가능한 경우에만 실제 방문 반응을 별도로 표시한다.

## 4.3 혼잡 관광지의 관심 분산

LOVV는 혼잡도가 높은 유명 관광지와 유사한 테마를 가진 주변 지역, 다른 시간대와 대체 장소를 제안한다.
사용자가 대체 여행지의 상세 정보를 클릭하고 일정에 포함한 비율을 통해 관심 분산 여부를 확인한다.

## 4.4 지역 단위 일정 확장

전통시장이나 축제 한 곳만 보여주는 대신 인근 문화자원, 산책로와 체험을 반일 또는 1일 일정으로 연결한다.
개별 장소 클릭이 지역 전체의 탐색과 체류 계획으로 확장되도록 돕는다.

## 4.5 공식 정보 활용과 신뢰 향상

관광지·축제 데이터가 단순 공개에서 끝나지 않고 추천 이유와 일정 구성에 사용된다.
기관명, 출처, 갱신일과 유효기간을 함께 표시해 사용자가 방문 전 최신 정보를 확인할 수 있게 한다.

# 5. 비재정 성과 측정

## 5.1 핵심 성과지표

| 단계 | 지표 | 산식·의미 |
| --- | --- | --- |
| 노출 | 이번 달 여행지 순노출 | 중복 사용자를 제거한 월간 지역 카드 노출 수 |
| 관심 | 상세 클릭률 | 여행지 상세 클릭/이번 달 여행지 순노출 |
| 탐색 | 콘텐츠 탐색률 | 장소·축제·추천 근거를 확인한 사용자/상세 클릭 사용자 |
| 계획 | 일정 생성률 | 해당 지역 일정 생성/상세 클릭 사용자 |
| 저장 | 일정 저장률 | 해당 지역 일정 저장/일정 생성 사용자 |
| 방문 준비 | 지도·교통·공식 링크 이동률 | 방문 준비 링크 클릭/상세 클릭 사용자 |
| 방문 의향 | 방문 고려 응답률 | 방문 의향 응답/응답 요청 사용자 |
| 방문 확인 | 선택적 방문 확인률 | 방문 완료 응답/사후 응답 사용자 |
| 분산 | 대체 여행지 선택률 | 대체 여행지 클릭 또는 일정 선택/대체 여행지 노출 |
| 정보 품질 | 유효 정보 비율 | 갱신 기준을 충족한 지역정보/전체 노출 지역정보 |

핵심 전환 흐름은 `순노출 → 상세 클릭 → 일정 생성·저장 → 지도·공식 링크 이동 → 방문 의향·선택적 방문 확인`으로 본다.

## 5.2 방문효과 해석 원칙

- 노출 수는 지역이 발견된 규모를 의미한다.
- 상세 클릭은 사용자가 지역에 관심을 보인 행동이며 실제 방문으로 간주하지 않는다.
- 일정 생성·저장과 지도·교통 링크 이동은 방문 준비 행동으로 본다.
- 실제 방문은 사용자의 선택적 사후 응답, 캠페인 QR 또는 기관의 비식별 집계처럼 확인 가능한 자료가 있을 때만 표시한다.
- 방문 확인 자료가 없으면 `방문효과`를 노출, 관심과 방문 준비 행동의 증가로 제한해 표현한다.
- 개인의 위치 이력이나 민감정보를 방문 확인 목적으로 수집하지 않는다.

## 5.3 기관 공유 리포트

월간 리포트는 다음 항목으로 구성한다.

- 이번 달 여행지별 순노출 수와 상세 클릭률
- 테마별 상세 탐색, 일정 생성·저장 수
- 지도·교통·공식 페이지 이동 수
- 대체 여행지 노출과 선택 비율
- 선택적으로 확인된 방문 의향과 방문 반응
- 관심은 높지만 공식 데이터가 부족한 테마
- 만료 예정 또는 갱신이 필요한 지역정보

리포트에는 매출, 세수, 경제 파급액, 투자 대비 효과 등 재정 지표를 포함하지 않는다.

## 6.2 추천 공정성

기관의 협력 여부나 자료 제공량만으로 목적지 추천 순위를 높여서는 안 된다.
목적지 추천은 사용자 조건, 테마 적합도, 계절성, 월별 기상 경향, 혼잡도와 데이터 근거를 기준으로 생성한다.

기관은 다음 권한을 갖지 않는다.

- 특정 목적지의 추천 순위 강제 상승
- 추천 이유의 사실관계 왜곡
- 검증되지 않은 축제·장소의 일정 강제 삽입
- 경쟁 지역의 추천 제외
- 사용자 개인 여행이력 열람

## 6.3 데이터·윤리 기준

- 공공데이터의 이용조건과 출처 표시 기준을 준수한다.
- 기관명, 출처, 갱신일과 정보 유효기간을 표시한다.
- 사용자 행동정보는 익명·집계 수준으로만 공유한다.
- 민감정보, 개인 위치 이력과 개인 여행이력을 기관에 제공하지 않는다.
- 방문 여부 피드백은 선택 사항으로 제공하고 미응답에 불이익을 주지 않는다.
- 종료·휴장·변경 정보를 신속히 반영하고 오래된 정보는 노출에서 제외한다.
- 협력기관 자료도 LOVV의 동일한 품질·검증 기준을 통과해야 한다.

# 7. 비영리 B2G 의사결정 원칙

1. B2G 협력의 목적은 지역 관광자원의 공정한 발견과 방문 기회 확대에 둔다.
2. `이번 달 여행지`는 월별·테마별 적합성과 공식 정보의 신뢰도를 기준으로 운영한다.
3. 기관의 참여 여부가 추천 순위나 사용자 선택을 왜곡하지 않게 한다.
4. 성과는 노출, 상세 클릭, 일정 행동, 공식 정보 이동과 확인 가능한 방문 반응으로 측정한다.
5. 클릭은 실제 방문과 구분하며, 방문 확인 근거가 있을 때만 방문 성과로 표시한다.
6. 예산, 과금, 수익, 계약금액과 경제효과 등 재정적 사유는 본 협력안의 판단 근거에서 제외한다.
7. 사용자 행동정보는 익명·집계 수준으로만 활용하고 개인 위치 이력은 수집하지 않는다.
8. 모든 협력 콘텐츠에는 출처, 갱신일과 검증 상태를 표시한다.

> 주의: 본 문서의 방문효과는 우선 지역 노출, 상세 클릭, 일정 생성·저장, 지도·공식 정보 이동과 선택적으로 확인된 방문 반응을 의미한다. 노출이나 클릭만으로 실제 방문 증가를 보장하거나 단정하지 않는다.
