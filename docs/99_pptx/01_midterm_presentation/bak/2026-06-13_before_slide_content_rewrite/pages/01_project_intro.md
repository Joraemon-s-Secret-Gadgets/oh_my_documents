**핵심 메시지 (화면 카피)**
서울·오사카 말고, 지금은 이곳

**서브 카피**
지도에서 흐릿한 곳, 여행에선 가장 빛나는 곳

**한 줄 정의 (로고 아래 작게, `Sub Text Gray`) — 신규**
한일 소도시를 찾아주는 대화형 여행 추천 서비스

**화면 구성**
- 좌측: 로고 타이포 "Lovv" + 메인 카피 + 서브 카피 + 한 줄 정의.
- 우측: Card 방식으로 Lovv 의미 적재.
  - **Lo / Local** — 소도시
  - **V / Vibe** — 분위기·감성
  - **V / Voyage** — 여행
- 표지 화면에서 `중간발표 / PoC` 표기는 제거한다.
- 발표날짜 `2026.06.16`과 발표자 `조동휘`는 우측 하단에 작게 배치한다.
- 배경 `Soft Beige`, 로고 `Main Orange`. 본문 텍스트 없음(시적 카피 + 한 줄 정의 + 의미 카드 + 발표 정보만).

**비즈니스 포인트 (작게)**
한일 소도시 여행 추천 / 인바운드·아웃바운드 양방향 타깃

**발표자 노트 (오프닝 ~15초 · 키워드·뉘앙스 큐)**
- 인사 → "저희가 만든 건 **Lovv**" · *(짧고 자신 있게)*
- "서울·오사카 **말고** 한·일 소도시" · 대화형 추천 · *('말고'에 살짝 힘)*
- "오늘 — 왜·어떻게 만들었나" · *(기대감만 열기)*
- ⚠ 소개만. 문제(오버투어리즘)는 S2에서 — 여기서 미리 꺼내지 X.
- (참고) 첫 질문은 "어디로?"가 아니라 "언제·어느 나라로?" — 도시를 이미 정했다고 가정하지 않는다(S5 컨셉①의 복선).

> 화면용/노트용: S1은 미니멀 유지. 화면 = 로고 + 카피 + 한 줄 정의 + Lovv 의미 카드 + 발표 정보. 단계 표기와 PoC 문구는 표지에서 제외한다. "어떻게 작동"은 S5·본론 몫.
