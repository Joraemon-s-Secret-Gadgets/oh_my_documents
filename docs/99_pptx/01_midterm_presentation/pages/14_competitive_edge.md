| 비교 기준 | 트리플 중심 | Lovv |
| --- | --- | --- |
| 시작점 | 도시를 정한 뒤 일정 생성 | 국가·시기·취향에서 소도시 발견 |
| 타깃 | 대도시·인기 여행지 중심 | 한일 소도시·축제 중심 |
| 추천 근거 | 일정·상품 연결 중심 | RAG 기반 이유·출처 제공 |
| 비즈니스 확장 | 예약·상품 판매 | 지역 분산·파트너 대시보드 가능성 |

## 발표자 노트

- 트리플을 경쟁자로 공격하기보다 사용 시작점이 다르다고 설명합니다.
- 트리플은 이미 정한 여행지를 잘 관리하는 데 강하고, Lovv는 목적지 발견 문제에 집중합니다.
