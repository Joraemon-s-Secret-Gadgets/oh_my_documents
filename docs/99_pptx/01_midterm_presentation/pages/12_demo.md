# 슬라이드 B6 — 데모

> 원본 위치: `../01_midterm_presentation.md`
> 상태: Slide Content
> 역할: PoC가 실제로 어떤 흐름으로 동작하는지 보여줌

## 화면 문구

**조건 입력에서 근거 있는 일정까지**

| 장면 | 보여줄 것 |
| --- | --- |
| 1 | 국가·시기·취향 입력 |
| 2 | 소도시 추천 후보 확인 |
| 3 | 추천 이유와 출처 확인 |
| 4 | 일정 생성 결과 확인 |
| 5 | 지도·상세 정보 이동 |

## 레이아웃

| 영역 | 내용 |
| --- | --- |
| 중앙 | 데모 영상 또는 캡처 3~5컷 |
| 하단 | 입력 -> 추천 -> 근거 -> 일정 타임라인 |

## 발표자 노트

- 지금까지 설명한 시스템 흐름을 실제 사용자 화면으로 보여줍니다.
- 데모에서는 추천 결과보다 “왜 추천됐는지”가 보이는 부분을 강조합니다.

## 제작 체크

- [ ] 영상이 없으면 캡처 5컷으로 대체한다.
- [ ] 데모 중 설명은 30초 내로 제한한다.
