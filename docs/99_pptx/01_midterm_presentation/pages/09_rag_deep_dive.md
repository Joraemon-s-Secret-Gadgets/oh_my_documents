> 역할: 검색 결과를 Planner가 쓸 수 있는 근거 패키지로 압축

## 화면 문구

**근거로 추천을 설명합니다**

| 단계 | 산출 |
| --- | --- |
| 후보 검색 | Vector + DB |
| 점수화 | 테마·시기·혼잡 |
| 근거 묶음 | 추천 이유 |
| 입력 고정 | Planner 입력 |

**Evidence Package**
선택 도시 · 추천 장소 · 예비 후보 · 검증 로그

## 발표자 노트

- Candidate Evidence는 "검색 결과 목록"이 아니라 Planner가 사용할 추천 근거 패키지다.
- 먼저 국가, 도시, 테마, 월 같은 metadata filter로 후보 범위를 줄이고, 의미 검색으로 사용자의 취향과 가까운 장소를 찾는다.
- 이후 테마 적합도, 월별 적합도, 혼잡 보조 지표, 일정 구성 가능성을 함께 보고 후보를 압축한다.
- 최종 패키지에는 선택 도시, 추천 장소, 예비 후보, 검증 로그가 들어간다.
- Planner는 이 패키지 안의 근거만 사용하므로 추천 이유와 일정 설명을 나중에 다시 확인할 수 있다.
