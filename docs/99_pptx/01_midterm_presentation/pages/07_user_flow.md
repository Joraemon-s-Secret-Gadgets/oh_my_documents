# 슬라이드 6 — 유저 플로우

> 원본 위치: `../01_midterm_presentation.md`
> 상태: Slide Content
> 역할: 사용자가 도시가 아니라 조건에서 시작한다는 차이를 설명

## 화면 문구

**도시를 몰라도 조건만 말하면 됩니다**

조건을 말한다 → 소도시를 받는다 → 일정으로 실행한다

| 단계 | 내용 |
| --- | --- |
| 조건을 말한다 | 일본 · 5월 · 조용한 바다 · 축제 |
| 소도시를 받는다 | 추천 도시 1곳 · 추천 이유 · 근거 뱃지 |
| 일정으로 실행한다 | 오전/오후 일정 · 지도 · 상세 링크 |

## 발표자 노트

- 도시를 이미 정했다고 가정하지 않는다.
- 조건 입력만으로 추천과 일정 실행까지 이어진다.
- 하단 설명 문구는 사용하지 않고, 카드 문구 자체로 흐름을 이해하게 한다.
