## 화면 문구

**오버투어리즘**

특정 관광지·시기에 수용 능력을 넘는 관광객이 몰려 주민 생활, 환경, 도시 기능, 지역 문화에 부담이 생기는 현상

| 사례 | 한 줄 설명 |
| --- | --- |
| 주민 생활 | 주거지 관광지화로 소음, 무단 촬영, 사생활 침해가 발생한다. |
| 환경 부담 | 쓰레기, 플라스틱, 생활폐기물 처리 비용이 증가한다. |
| 도시 기능 | 교통, 숙박, 공공시설 수용 능력이 성수기에 압박받는다. |
| 지역 문화 | SNS 명소화와 단기 소비형 방문이 지역 고유성을 약화시킨다. |
