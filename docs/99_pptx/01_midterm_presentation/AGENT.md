# Agent Instructions for midterm presentation

이 폴더는 로브(Lovv) 중간발표 자료의 대표 Markdown, 페이지별 슬라이드 문서, 발표 규칙과 참고 자료를 관리한다.

## Primary Document

대표 문서는 `01_midterm_presentation.md`다. 발표 전체 흐름, 디자인 기준, 슬라이드별 핵심 메시지는 이 파일을 기준으로 관리한다.

## Document Structure

현재 하위 문서는 다음과 같다.

- `01_midterm_presentation.md`: 중간발표 대표 Markdown
- `pages/*.md`: 슬라이드 한 장 단위의 화면 문구와 발표자 노트
- `pages/slide_map.md`: 페이지별 슬라이드 맵
- `meta/*.md`: 발표 작성 규칙과 리뷰 피드백
- `references/*.md`: 발표 구성에 참고한 이전 원본과 제안 문서
- `archive/`: 정리 완료된 과거 작업 문서
- `bak/`: 로컬 백업 전용 폴더

## Editing Rules

- 발표 전체 흐름을 바꾸면 먼저 `01_midterm_presentation.md`에 반영한다.
- 개별 슬라이드 화면 문구나 발표자 노트를 바꾸면 대응하는 `pages/*.md`를 함께 확인한다.
- `pages/slide_map.md`는 페이지 파일의 순서와 용도를 설명하는 보조 문서로 유지한다.
- `meta/`와 `references/`는 근거 자료이며, 대표 문서와 충돌하는 정본으로 만들지 않는다.
- `bak/`는 GitHub 업로드 대상이 아니며, 현재 원본 판단에는 사용하지 않는다.
