ISSUE / TROUBLESHOOTING

그래프DB 직접 도입 대신 Lambda로 관계 탐색 기능을 구현한다

1. 이슈

현재 PoC 관계는 대부분 1~2-hop 수준

2. 비용

Neptune Serverless 시작 용량 1 NCU, 고정비 비중 큼

3. 판단

Lambda로 그래프DB 유사 기능을 구현

4. 승격 기준

3-hop+ 임의 경로 탐색·실시간 그래프 쓰기 필요 시 도입

관계 탐색은 Lambda 기반 재생성 가능 보조 기능으로 두고, Neptune은 승격 옵션으로 남긴다.
