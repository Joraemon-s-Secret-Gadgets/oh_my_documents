# 9. 저장·피드백·마이페이지 흐름

| 기능 | PoC | Production |
| --- | --- | --- |
| 일정 저장 | 로컬 스토리지 | `POST /me/itineraries`로 로그인 계정 서버 저장 |
| 온보딩 선호 | 로컬 스토리지 | `PUT /me/preferences`로 사용자 프로필 저장 |
| 좋아요/싫어요 | 로컬 스토리지 | `POST /me/feedback`으로 추천 피드백 테이블 저장 |
| 마이페이지 | 기본 조회 | 저장 일정 상세, 선호 재설정, 피드백 이력, 삭제 기능 |
| 대화 로그 | 전문 저장 금지 | 전문 저장 금지. 구조화된 선호와 최종 일정만 저장 |
