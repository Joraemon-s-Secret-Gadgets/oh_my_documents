# Agent Instructions for Aha Moment docs

이 폴더는 로브(Lovv)의 Aha Moment 설계 문서와 화면 단계별 보조 문서를 관리한다.

## Document Structure

대표 문서는 `95_aha_moment.md`다. Aha 정의, 추천 여정 funnel, cold start 전략, 재방문 개인화 원칙은 이 파일을 기준으로 관리한다.

현재 하위 문서는 다음과 같다.

- `theme-detail.md`: THEME DETAIL 화면의 추천 기준·납득 경험 보조 설계
- `followup-requery.md`: 후속 질의와 재추천 흐름 보조 설계
- `plan-detail.md`: PLAN DETAIL 화면의 일정 생성·저장 경험 보조 설계
- `aha-card-slot.md`: 메인 카드 슬롯의 기본·개인화 노출 보조 설계
- `tracking-plan.md`: Aha 관련 이벤트와 추적 항목 보조 설계

## Editing Rules

- Aha Moment의 핵심 정의나 funnel 단계를 바꿀 때는 먼저 `95_aha_moment.md`에 반영한다.
- 화면별 보조 문서는 대표 문서의 세부 구현 단위로 작성하고, 대표 문서와 충돌하는 독립 정본으로 만들지 않는다.
- 보조 문서의 내용이 확정되면 대표 문서에 요약 반영한다.
- 문서 버전, 문서 상태, 적용 범위를 상단 메타데이터에 유지한다.
