# Agent Instructions for 93_B2B

이 폴더는 로브(Lovv)의 B2B 제휴 전략, 수익 모델, 파트너별 가치 제안과 사업성 검증 계획을 관리한다.

## Editing Rules

- 대표 문서는 `93_B2B.md`이다.
- 공개된 공식 자료와 내부 가정을 구분하고, 수수료율·전환율·거래액 등 변동 가능한 수치는 기준일과 출처를 명시한다.
- 제휴사의 기능을 과도하게 단순화하지 않는다. 확인된 기존 기능과 로브가 추가로 만드는 가치를 분리한다.
- 예약 건수와 실제 이용 완료 건수, 총 예약액(GMV)과 로브 매출을 혼동하지 않는다.
- 손익 추정치는 공식 확정값이 아닌 시나리오임을 표시하고 산식을 함께 제공한다.
- 제휴 링크, 쿠키, 딥링크, API, 이미지 사용, 광고성 표시 등 각 파트너의 운영 정책을 출시 전에 재검증한다.
- 내용 변경은 Markdown 원본에 먼저 반영하고, 공유용 HTML은 생성물로 취급한다.
