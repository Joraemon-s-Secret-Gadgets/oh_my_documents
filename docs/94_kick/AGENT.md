# Agent Instructions for 94_kick

이 폴더는 로브(Lovv)의 제품 선택을 촉발하는 kick 가설과 검증 계획을 관리한다.

## Editing Rules

- 대표 문서는 `94_kick.md`다.
- kick은 단순한 흥미나 기능 추가가 아니라 사용자가 추천을 실제 일정 선택·복제·저장으로 전환하게 만드는 장치로 정의한다.
- `docs/95_aha_moment/`의 발견·납득·실행가능성 설계와 모순되지 않게 작성한다.
- 연령·성별 등 민감한 프로필은 선택 입력, 구간화, 공개 동의, 최소 집단 크기 원칙을 지킨다.
- 가설 수치와 확정 수치를 구분하고, 검증 전 수치는 반드시 가설로 표시한다.
- 내용 변경은 Markdown 원본에 먼저 반영하고, 공유용 HTML은 생성물로 취급한다.
