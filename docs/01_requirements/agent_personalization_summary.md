## 제약

- 대화 로그 전문은 저장 대상에서 제외한다.
- PoC 개인화 데이터는 브라우저 로컬 스토리지 중심으로 관리한다.
- Production 단계에서는 로그인 사용자 계정 기반으로 누적·조회·삭제할 수 있도록 확장한다.
- 추천 다시 받기 요청 시 직전 후보는 제외 또는 감점한다.
