# 로컬루트 신규 테마 제안 — 공예 · 장인 체험 (Craft & Maker)

> 기존 6개 테마(온천·바다·역사·미식·자연·예술)에 더할 신규 테마 제안서입니다.
> 한·일 양국 데이터로 받쳐지며, 기존 예술 테마와 결이 겹치지 않도록 분리됩니다.

---

## 개요

보기만 하는 여행이 아니라 **직접 만드는 여행**입니다. 도자기·한지·염색·칠기·유리공예 등 소도시가 산지인 전통 공예를 직접 체험합니다. 소도시가 공예 산지인 경우가 많아 궁합이 좋고, "체험형"이라 기존 관광과 차별화됩니다. 예술 테마와 겹쳐 보이지만 **감상(예술) vs 직접 제작(공예)** 으로 명확히 분리됩니다.

| 항목 | 내용 |
|---|---|
| 테마 ID | `craft` |
| 한 줄 정의 | 전통 공예를 직접 만들어보는 체험형 소도시 여정 |
| 기존 예술 테마와 차이 | 예술은 "감상", 공예는 "직접 제작" — 손으로 만드는 체험이 핵심 |
| 한국 대표지 | 여주·이천(도자기), 전주(한지), 안동(전통 공예), 강진(청자) |
| 일본 대표지 | 마시코·아리타(도자기), 가나자와(금박·칠기), 공예 산지 소도시 |
| 계절 연계 | 사계절 가능(실내 체험 중심), 우천에도 적합 |

---

## 사용자 시나리오

### 시나리오 C-01 · 연인의 도자기 만들기 당일 코스

> 사용자가 "연인과 특별한 걸 직접 만들어보는 가까운 여행"을 요청한다. 시스템은 공예 테마로 분류하고, 도자기 체험 공방이 있는 소도시를 추천한다. 공예 체험 + 주변 식당·산책 코스를 묶어 제안하며, 체험 소요 시간과 완성품을 받는 방법(현장 / 추후 배송)을 함께 안내한다. 비 오는 날에도 실내 체험이라 적합함을 표시한다.

### 시나리오 C-02 · 한국 여행자의 일본 도자기 마을 순례

> 한국 사용자가 가을 일본 여행에서 "전통 도자기를 직접 빚어보고 싶다"고 입력한다. 시스템은 공예 테마 + 가을 적합도로 도자기 산지 소도시를 추천하고, 공방 체험 정보와 도자기 마을 산책 코스를 함께 제시한다. 마침 열리는 도자기 축제(월·계절 단위)가 있으면 함께 안내한다.

---

## 유저 스토리

- **US-C1** — 사용자로서, 나는 직접 만들어보는 체험이 있는 소도시를 추천받고 싶다. 그래야 구경만 하는 여행이 아닌 손으로 남기는 경험을 할 수 있다.
- **US-C2** — 사용자로서, 나는 공예 체험의 소요 시간과 난이도를 미리 알고 싶다. 그래야 일정과 동행자에 맞는 체험을 고를 수 있다.
- **US-C3** — 여행자로서, 나는 만든 작품을 받는 방법(현장 수령 / 추후 배송)을 알고 싶다. 그래야 일정과 귀국 계획에 맞춰 결정할 수 있다.
- **US-C4** — 사용자로서, 나는 비 오는 날에도 즐길 수 있는지 알고 싶다. 그래야 날씨와 무관하게 일정을 확정할 수 있다.
- **US-C5** — 사용자로서, 나는 공예 체험과 그 지역 도자기 마을·축제를 한 여정으로 묶어 받고 싶다. 그래야 체험과 주변 관광을 따로 계획하지 않아도 된다.

## 추천 시 포함 항목

- 공예 종류(도자기 / 한지 / 염색 / 칠기 / 유리 등)
- 체험 소요 시간 · 난이도 · 예약 필요 여부
- 완성품 수령 방법(현장 / 배송)
- 우천 적합성(실내 체험 여부)
- 주변 공예 마을·축제 연계 정보

## 기존 구조 반영 방식

| 반영 위치 | 내용 |
|---|---|
| 테마 가중치 | 기존 6개 → 추가 (`craft`) |
| 소도시 DB | 각 소도시의 `themes` 객체에 `craft` 점수 추가 |
| 계절 연계 | 사계절 가능(실내 체험 중심) |
| 우천 적합성 | 실내 체험이라 우천 적합 태그로 강점 부각 |
| 동행 유형 | 연인 / 가족 / 친구 적합 |
| 양국 데이터 | 한국(도자기·한지) · 일본(도자기·칠기) 모두 확보 가능 |
