# 3. 서비스 개요

## 3.1 서비스 소개

로브(Lovv)는 오버투어리즘 회피를 목적으로 국내 및 일본의 비주류 소도시 1곳을 사용자의 취향과 여행 시기에 맞는 집중 일정으로 추천하는 대화형 챗봇 서비스다.
사용자가 자연어로 취향과 여행 시기를 입력하거나 지도에서 소도시 마커를 선택하면, 멀티스텝 에이전트가 조건을 테마 또는 추천 진입점으로 해석하고 RAG로 검색한 소도시 데이터에 근거하여 적합한 소도시 1곳과 일별 세부 일정을 추천 이유와 함께 제공한다.
한국은 한국 소도시만, 일본은 일본 소도시만 추천하는 나라별 독립 트랙으로 운영된다.

서비스명 로브(Lovv)는 Local(소도시)과 Vibe(분위기·감성), 또는 Voyage(항해·여행)의 의미를 결합한 이름이다.
일반적인 Love와 달리 알파벳 v를 겹쳐 사용해 감성적인 여행 서비스이면서도 트렌디한 스타트업·테크 서비스의 인상을 전달한다.

추천 시 여행 시기(월)에 빛나는 소도시와 사용자가 포함을 선택한 축제·행사를 일정에 결합하며, 각 결과에는 Google Maps, Kakao Maps, Yahoo Japan, WeatherAPI의 4개 플랫폼 API가 역할 분담하여 지도·장소 상세·현재 날씨·링크 정보를 보완 제공한다.
지역 문화 체험, 스키 액티비티 같은 P1~P4 확장 테마와 월별 기상 경향을 기준으로 한 직접 질의형 추천도 같은 추천 엔진 안에서 처리한다.
별 관측, 치유·웰니스, 예산 역산 추천은 이번 프로젝트 대상에서 제외하고 향후 단계에서 검토한다.

## 3.2 핵심 가치 제안

| 가치 항목 | 설명 |
| --- | --- |
| 소도시 집중 일정 추천 | 기본·확장 테마와 사용자 조건에 맞는 소도시 1곳을 선정하고 당일치기~4박5일 일정을 설계해 제공 |
| 계절·축제 연동 | 여행 시기에 빛나는 소도시와 사용자가 포함을 선택한 축제·행사를 일정에 결합 |
| 직접 질의형 추천 | 사용자가 여행 시기(월)를 바로 말하면 테마 선택 없이 월별 기상 경향이 좋은 소도시를 역으로 제안. 예산 역산은 향후 검토 |
| 지도 기반 진입 | 메인 지도에서 국가·월·테마로 소도시를 필터링하고 마커 클릭으로 추천을 시작 |
| 개인화 루프 | 온보딩 선호, 저장 일정, 북마크, 실제 방문 기록, 방문 소도시 별점, 좋아요/싫어요 피드백을 다음 추천 조건과 후보 랭킹에 반영 |
| 혼잡 회피형 추천 | 단순 인기순이 아닌 혼잡도 낮은 대체 여행지를 우선 추천 |
| RAG 기반 정확성 | 직접 구축한 소도시 데이터를 검색·근거로 활용해 환각 없는 사실 기반 추천 |
| 멀티스텝 에이전트 | 분류→검색→선정→일정 구성을 단계별로 추론하여 추천 품질 확보 |
| 국내·일본 통합 | 한국 소도시와 일본 소도시를 단일 서비스에서 나라별 트랙으로 탐색 |

## 3.3 타겟 사용자

| 구분 | 설명 |
| --- | --- |
| 주 타겟 | 국내외 여행을 즐기지만 혼잡한 인기 여행지를 피하고 싶은 20~40대 성인 |
| 부 타겟 | 일본 여행 경험이 있어 도쿄·오사카 외 소도시를 탐색하는 여행자 |
| 공통 특성 | 자연어 대화에 익숙하고, 스마트폰 중심의 여행 정보 탐색 습관 보유 |

## 3.4 서비스 흐름 요약

| 단계 | 입력 | 처리 / 출력 |
| --- | --- | --- |
| 1 | 진입 및 조건 파악 | 온보딩 선호 프로필이 기본값으로 적용됨. 사용자는 지도 마커 클릭 또는 챗봇 대화로 진입하며 국가·여행 시기(월)·테마를 파악 |
| 2 | 진입점 분류 | 지도 마커 클릭(목적지+일정 유형+축제 포함 여부) 또는 챗봇 발화(테마·시기·월별 기상 경향 직접 질의)로 분류 |
| 3 | RAG 검색·후보 선정 | 테마+계절 적합도+월별 기상 경향 데이터를 기준으로 소도시 1곳을 선정 |
| 4 | 일정 구성 | 선택한 일정 유형(당일치기~4박5일) 기준 일별 세부 일정 생성. 축제 포함 여부에 따라 축제·행사 정보를 일정에 결합 |
| 5 | 상세 정보 조회 | 드로어 패널에서 지도·장소 정보·날씨·외부 링크 확인 |
| 6 | 외부 연동 | 숙소 검색 링크, Kakao / Yahoo Japan 딥링크로 추가 정보 확인 |
