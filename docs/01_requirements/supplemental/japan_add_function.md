## 1. 기준 방향

`로컬루트 Japan`은 도쿄·오사카·교토 같은 유명 관광지 대신, 사용자의 취향·일정·예산·계절·접근성을 기준으로 일본 소도시를 추천하는 챗봇 서비스로 확장하는 것이 적합하다.

핵심 컨셉은 다음과 같다.

> 유명 관광지 대신, 내 취향과 일정에 맞는 일본 소도시를 추천해주는 여행 챗봇

---

## 2. 일본 기준 핵심 기능 우선순위

| 우선순위 | 기능 | 설명 |
|---|---|---|
| 1 | 대도시 대체 추천 | 교토·오사카·도쿄 대신 갈 수 있는 유사 분위기의 소도시 추천 |
| 2 | JR/공항 접근성 표시 | 주요 도시 기준 이동시간, 환승 횟수, JR 패스 사용 가능 여부 표시 |
| 3 | 일본 여행 스타일 프리셋 | 온천, 료칸, 성하마을, 바다마을, 기차여행 등 선택형 추천 |
| 4 | 계절별 추천 | 벚꽃, 단풍, 설경, 온천, 축제 등 계절 조건 반영 |
| 5 | 일본어 검색 키워드 제공 | 실제 검색에 유용한 일본어 키워드 제공 |
| 6 | 숙박·맛집 딥링크 | じゃらん, 楽天トラベル, 食べログ 등 외부 플랫폼 연결 |
| 7 | 혼잡 회피 근거 표시 | 왜 덜 붐비는지, 어떤 시기에 가야 좋은지 설명 |
| 8 | 교통패스 추천 | JR 패스, 지역 패스 등 비용 절감 정보 제공 |
| 9 | 여행 경험 수준 분기 | 처음 일본 / N회차 일본 여행자에 따라 추천 기준 변경 |
| 10 | 일정 자동 생성 | 당일치기, 1박2일, 2박3일 코스 생성 |

## 3. 추가 기능 상세

## 3.1 대도시 대체 추천

사용자가 유명 관광지를 입력하면 유사한 분위기의 덜 붐비는 소도시를 추천한다.

### 예시

| 입력 | 대체 추천 |
|---|---|
| 교토 말고 조용한 전통 마을 | 가나자와, 마쓰에, 다카야마 |
| 오사카 말고 로컬 감성 도시 | 구라시키, 오노미치, 와카야마 |
| 도쿄 근교 조용한 여행 | 가와고에, 닛코, 치치부 |
| 후쿠오카 근교 온천/소도시 | 유후인, 히타, 가라쓰 |

### 요구사항 ID

```text
FR-JP-001 사용자가 일본 유명 관광지를 입력하면 유사한 분위기의 소도시 대체지를 추천한다.
```

## 3.2 JR/신칸센/공항 접근성 표시

일본 소도시는 이동 난이도가 서비스 만족도에 큰 영향을 준다. 추천 결과에는 접근성을 반드시 포함하는 것이 좋다.

### 표시 항목

- 도쿄 출발 기준 이동시간
- 오사카 출발 기준 이동시간
- 교토 출발 기준 이동시간
- 후쿠오카 출발 기준 이동시간
- 나고야 출발 기준 이동시간
- 신칸센 이용 가능 여부
- JR 패스 사용 가능 여부
- 환승 횟수
- 가장 가까운 공항
- 뚜벅이 여행 가능 여부

### 예시

```text
가나자와
- 도쿄 출발: 호쿠리쿠 신칸센 약 2시간 30분
- 오사카 출발: 특급+신칸센 약 2시간 40분
- JR 패스: 사용 가능
- 환승 난이도: 쉬움
- 뚜벅이 여행: 가능
```

### 요구사항 ID

```text
FR-JP-002 일본 목적지는 주요 도시 기준 이동시간, 환승 횟수, JR 패스 활용 여부를 표시한다.
```

## 3.3 일본 여행 스타일 프리셋

사용자가 긴 문장을 입력하지 않아도 원하는 여행 유형을 빠르게 선택할 수 있게 한다.

- 료칸 숙박

- 바다 마을
- 산악 소도시
- 로컬 시장
- 애니/성지순례
- 기차 여행
- 설경 여행
- 단풍 여행
- 벚꽃 여행
- 혼자 조용히 쉬기
- 부모님 동반
- 커플 감성 여행
- 사진 여행

### 요구사항 ID

```text
FR-JP-003 사용자는 일본 여행 스타일 프리셋을 선택할 수 있다.
```

## 3.4 계절별 추천

일본 여행은 계절성이 강하므로 계절 정보를 추천 로직에 반영해야 한다.

### 계절 태그 예시

| 계절 | 추천 기준 |
|---|---|
| 봄 | 벚꽃, 신록, 온화한 날씨 |
| 여름 | 해안, 섬, 축제, 고원 |
| 가을 | 단풍, 온천, 산악 도시 |
| 겨울 | 설경, 료칸, 온천, 일루미네이션 |

### 예시

```text
12월 추천
1. 긴잔온천 — 설경과 온천 분위기
2. 다카야마 — 겨울 전통거리 감성
3. 가나자와 — 겨울 겐로쿠엔과 미식 여행
```

### 요구사항 ID

```text
FR-JP-004 일본 목적지는 계절별 추천 시기와 비추천 시기를 표시한다.
```

## 3.5 일본어 검색 키워드 제공

일본 현지 정보는 한국어보다 일본어 검색 결과가 더 풍부하다. 목적지별로 바로 검색 가능한 일본어 키워드를 제공하면 실사용성이 높아진다.

### 예시

```text
가나자와 검색 키워드
- 金沢 観光
- 金沢 モデルコース
- 金沢 カフェ
- 金沢 穴場
- 金沢 ひとり旅
```

### 요구사항 ID

```text
FR-JP-005 목적지별 일본어 검색 키워드를 제공한다.
```

```text
FR-JP-006 일본 목적지 상세에서 숙박 플랫폼 딥링크를 제공한다.
```

## 3.7 교통패스 추천

일본 지방 여행은 교통패스 활용 여부에 따라 비용 차이가 크다. 목적지별 사용 가능한 패스를 함께 추천하면 서비스 완성도가 높아진다.

### 예시 패스

- JR Pass
- Hokuriku Arch Pass
- JR East Pass
- JR West Kansai Wide Area Pass
- JR Kyushu Pass
- Shikoku Rail Pass
- San'in Okayama Area Pass

### 표시 예시

```text
추천 패스: Hokuriku Arch Pass
적합 구간: 도쿄/오사카 → 가나자와 → 도야마
절약 가능성: 높음
```

### 요구사항 ID

```text
FR-JP-008 목적지별 사용 가능한 교통패스 정보를 제공한다.
```

## 3.8 혼잡 회피 근거 표시

서비스의 핵심 가치가 오버투어리즘 회피이므로, 왜 해당 목적지가 덜 붐비는지 설명해야 한다.

### 근거 예시

- 교토/오사카 대비 외국인 관광객 집중도가 낮음
- 주요 단체 관광 루트와 거리가 있음
- 숙박 밀집도가 낮음
- 리뷰 수와 검색량이 상대적으로 낮음
- 평일 오전 방문 시 여유로움
- 동선 분산이 쉬움

### 표시 예시

```text
혼잡 회피 근거
- 교토 대비 외국인 관광객 밀도가 낮음
- 주요 관광 루트에서 벗어난 지역
- 평일 방문 시 여유로운 일정 가능
```

### 요구사항 ID

```text
FR-JP-009 일본 목적지는 혼잡 회피 근거를 최소 2개 이상 표시한다.
```

## 3.9 일본 여행 경험 수준 분기

처음 일본을 가는 사람과 N회차 여행자는 추천 기준이 다르다.

### 처음 일본 여행자

- 교통이 쉬운 지역 우선
- 한국어 정보가 많은 지역 우선
- 대도시에서 접근 쉬운 소도시 우선
- 여행 실패 가능성이 낮은 목적지 추천

### N회차 일본 여행자

- 덜 알려진 지역 추천
- 환승 난이도 어느 정도 허용
- 로컬성 높은 지역 추천
- 관광객이 적은 지역 우선

### 요구사항 ID

```text
FR-JP-010 사용자의 일본 여행 경험 수준에 따라 추천 기준을 다르게 적용한다.
```

## 3.10 일정 자동 생성

목적지만 추천하면 사용자가 다시 일정을 짜야 한다. 목적지 추천 이후 일정까지 이어지면 서비스 가치가 커진다.

### 일정 유형

- 당일치기
- 1박 2일
- 2박 3일
- 온천 중심 코스
- 맛집 중심 코스
- 기차 여행 코스
- 부모님 동반 여유 코스

### 요구사항 ID

```text
FR-JP-011 추천 목적지 기반 여행 일정을 자동 생성한다.
FR-JP-012 사용자는 당일치기, 1박2일, 2박3일 일정을 선택할 수 있다.
```

```text
FR-JP-004 일본 목적지는 계절별 추천 시기와 비추천 시기를 표시한다.

FR-JP-006 일본 목적지 상세에서 숙박 플랫폼 딥링크를 제공한다.

FR-JP-010 사용자의 일본 여행 경험 수준에 따라 추천 기준을 다르게 적용한다.
FR-JP-011 추천 목적지 기반 여행 일정을 자동 생성한다.
FR-JP-012 사용자는 당일치기, 1박2일, 2박3일 일정을 선택할 수 있다.
```

## 8. 최종 추천

일본 기준으로 기능을 추가한다면 가장 먼저 넣을 기능은 다음 5개다.

1. 대도시 대체 추천
2. JR/공항 접근성 표시
3. 일본 여행 스타일 프리셋
4. 계절별 추천
5. 일본어 검색 키워드 제공

이 5개가 들어가면 로컬루트는 단순 여행 챗봇이 아니라, `일본 소도시 특화 추천 서비스`로 정체성이 명확해진다.
