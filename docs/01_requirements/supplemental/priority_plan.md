## 우선순위 설정

PoC 단계는 P1 구현을 핵심 범위로 한다.
P2는 PoC에서 설계까지 완료하고, 일정 여유가 있을 경우 일부 구현한다.
P3는 PoC에서 설계 산출물까지만 정리한다.
P4는 Production 단계에서 구현한다.
P5는 이번 프로젝트 대상에서 제외하고 향후 검토한다.

| 우선순위 | 기능 범위 | PoC | Production | 상태 |
| --- | --- | --- | --- | --- |
| P1 | 오버투어리즘 회피, 한일 축제 기반 여정 추천, 월별 기상 경향 기반 추천 진입 | 구현 | 고도화 | 핵심 |
| P2 | 일본어 검색 키워드 제공 | 설계 | 구현 | 선택 구현 |
| P3 | 철도 패스 기반 소도시 여행 추천, 체험형 여행 추천, K-Drama 촬영지 탐색, 애니메이션 성지순례 | 설계 | 구현 | 설계 |
| P4 | 액티비티(스키) 여행 추천 | 정의 | 구현 | Production |
| P5 | 별 관측 조건 기반 여행지 추천, 치유/웰니스 여행 추천, 예산 역산 추천 | 제외 | 검토 | 제외 |

---

##### 현재 기획 범위는 P1 중심입니다.
- P2 이상 기능은 핵심 AI 추천 경험(P1)이 완성된 이후 진입합니다.
- P1 단계에서 recommendation flow, personalization, ranking, explainability가 충분히 검증되지 않으면 P2 구현을 시작하지 않습니다.
- P5는 향후 단계에서 검토합니다.
