# 로컬루트 신규 테마 제안 — 치유 · 웰니스 (Healing & Wellness)

> 문서 성격: 보조 Markdown
> 대표 문서: `../01_requirements.md`

> 기존 6개 테마(온천·바다·역사·미식·자연·예술)에 더할 신규 테마 제안서입니다.
> 한·일 양국 데이터로 받쳐지며, 기존 온천 테마와 결이 겹치지 않도록 분리됩니다.

---

## 개요

온천 테마를 "몸과 마음의 회복"이라는 색으로 한 단계 확장한 테마입니다. 단순 온천을 넘어 **숲 치유(삼림욕) · 사찰 템플스테이 · 요가/명상 리트릿**을 아우릅니다. 번아웃 시대에 "쉬러 가는 소도시" 수요가 확실하며, 일본은 온천·젠(禅) 문화, 한국은 템플스테이·치유의숲 인프라가 있어 양국 모두 데이터 확보가 가능합니다.

| 항목 | 내용 |
|---|---|
| 테마 ID | `wellness` |
| 한 줄 정의 | 몸과 마음의 회복을 목적으로 한 저자극·휴식형 소도시 여정 |
| 기존 온천 테마와 차이 | 온천은 "시설", 웰니스는 "회복 경험" — 숲·사찰·명상까지 포괄 |
| 한국 대표지 | 강원 치유의숲, 산사 템플스테이(순천·하동), 변산·지리산 자락 |
| 일본 대표지 | 온천 소도시(유후인·구로카와), 젠 사찰, 숲 테라피 인증지 |
| 계절 연계 | 사계절 가능, 특히 가을·겨울 적합도 높음 |

## 사용자 시나리오

### 시나리오 W-01 · 번아웃 직장인의 혼자 회복 여행

> 30대 직장인 사용자가 극심한 피로를 느끼며 "아무 생각 없이 며칠 쉬고 오고 싶다"고 입력한다. 시스템은 웰니스 테마로 분류하고, 자극이 적고 한적한 소도시를 우선 선별한다. 온천 + 숲 산책 + 사찰 템플스테이를 묶은 2박 3일 여정을 제안하며, "사람이 적고 조용한 시기"를 함께 안내한다. 사용자는 휴대폰을 내려놓고 회복에 집중할 수 있는 코스를 받는다.

### 시나리오 W-02 · 일본 여행자의 한국 템플스테이 체험

> 일본 사용자가 봄철 한국 여행을 계획하며 "한국의 전통 사찰에서 머무는 조용한 경험"을 요청한다. 시스템은 웰니스 테마 + 봄 적합도로 산사 소도시를 추천하고, 템플스테이 운영 정보와 주변 치유의숲을 함께 제시한다. 고유명사(사찰명)는 원어와 현지 표기를 병기한다.

---

## 유저 스토리

- **US-W1** — 지친 사용자로서, 나는 "쉬고 싶다"는 막연한 표현만으로도 회복형 소도시를 추천받고 싶다. 그래야 무엇을 검색할지 몰라도 휴식 여행을 시작할 수 있다.
- **US-W2** — 1인 여행자로서, 나는 혼자서도 부담 없이 머물 수 있는 조용한 곳을 추천받고 싶다. 그래야 안전하고 편안하게 회복에 집중할 수 있다.
- **US-W3** — 사용자로서, 나는 온천·숲·사찰·명상이 하나의 여정으로 묶인 코스를 받고 싶다. 그래야 회복 활동을 따로 찾아 조합하지 않아도 된다.
- **US-W4** — 외국인 여행자로서, 나는 템플스테이·치유의숲 같은 현지 회복 문화의 운영 정보를 명확히 알고 싶다. 그래야 예약·참여 방법을 헤매지 않는다.
- **US-W5** — 사용자로서, 나는 "사람이 적고 조용한 시기"를 함께 안내받고 싶다. 그래야 회복이라는 목적에 맞게 한적한 때 방문할 수 있다.

---

## 추천 시 포함 항목

- 회복 활동 구성(온천 / 숲 치유 / 템플스테이 / 명상 중 해당 요소)
- 한적함·저자극 정도 (혼잡 회피 점수 연계)
- 사찰·치유의숲 운영 정보 및 참여 방법
- 추천 이유 + 주변 휴식 코스
- 적합한 동행 유형(혼자 / 연인 / 부모님과)

## 기존 구조 반영 방식

| 반영 위치 | 내용 |
|---|---|
| 테마 가중치 | 기존 6개 → 추가 (`wellness`) |
| 소도시 DB | 각 소도시의 `themes` 객체에 `wellness` 점수 추가 |
| 계절 연계 | 사계절 가능, 가을·겨울 강세 |
| 동행 유형 | 혼자 / 부모님과 적합 |
| 양국 데이터 | 한국(템플스테이·치유의숲) · 일본(온천·젠) 모두 확보 가능 |
