# 로컬루트 신규 테마 제안 — 사계절 액티비티 (Seasonal Activity)

> 문서 성격: 보조 Markdown
> 대표 문서: `../01_requirements.md`

> 기존 6개 테마(온천·바다·역사·미식·자연·예술)에 더할 신규 테마 제안서입니다.
> 한·일 양국 데이터로 받쳐지며, 계절축과 곱해져 "언제 × 무엇을 한다"는 강력한 조합을 만듭니다.

---

## 개요

스키 하나에 그치지 않고 **계절별 액티비티**로 일반화한 테마입니다. 겨울 스키·보드, 여름 서핑·계곡·다이빙, 봄가을 트레킹·자전거·패러글라이딩을 아우릅니다. 우리 계절 시스템과 곱해지면 "여름엔 이 소도시에서 서핑, 겨울엔 저기서 스키"처럼 **같은 사용자가 계절에 따라 다른 추천을 받는** 구조가 됩니다. 액티비티는 여행 동기가 확실하고, 소도시(양양 서핑, 강원 스키)와 궁합이 좋습니다. 기존 테마가 모두 정적·관조형인 데 비해, 이 테마는 **동적·체험형** 축을 더해 "쉬는 여행 vs 노는 여행"을 모두 커버합니다.

| 항목 | 내용 |
|---|---|
| 테마 ID | `activity` |
| 한 줄 정의 | 계절에 맞는 야외 액티비티를 즐기는 체험형 소도시 여정 |
| 기존 자연 테마와 차이 | 자연은 "감상·휴식", 액티비티는 "직접 몸으로 즐기는 레저" |
| 계절별 구성 | 겨울=스키·보드 / 여름=서핑·계곡·다이빙 / 봄·가을=트레킹·자전거·패러글라이딩 |
| 한국 대표지 | 평창·정선(스키), 양양(서핑), 동강(래프팅), 단양(패러글라이딩) |
| 일본 대표지 | 니세코·하쿠바(스키), 쇼난·미야자키(서핑), 산악 트레킹 소도시 |
| 계절 연계 | **계절축과 직접 곱해지는 테마** — 시기에 따라 추천 액티비티가 바뀜 |

---

## 사용자 시나리오

### 시나리오 A-01 · 겨울, 친구와 스키 여행

> 사용자가 1월에 "친구들과 스키 타러 갈 소도시"를 요청한다. 시스템은 액티비티 테마 + 겨울 적합도로 스키장이 있는 소도시를 추천하고, 슬로프 규모·초중급 적합성과 함께 온천을 곁들인 "스키 + 온천" 여정을 제안한다. 같은 사용자가 여름에 다시 방문하면 동일 테마라도 서핑·계곡 소도시를 추천받는다.

### 시나리오 A-02 · 일본 여행자의 여름 한국 서핑 여행

> 일본 사용자가 여름철 한국 여행에서 "바다에서 서핑을 배우고 즐기고 싶다"고 입력한다. 시스템은 액티비티 테마 + 여름 적합도로 서핑 소도시(양양 등)를 추천하고, 초보 강습 가능 여부·장비 대여 정보와 주변 해변 코스를 함께 제시한다. 적합한 시기(파도·수온)를 월·계절 단위로 안내한다.

---

## 유저 스토리

- **US-A1** — 액티비티를 즐기는 사용자로서, 나는 "지금 계절에 할 수 있는" 레저 소도시를 추천받고 싶다. 그래야 시기에 맞지 않는 액티비티를 헛되이 찾지 않는다.
- **US-A2** — 사용자로서, 나는 같은 곳이라도 계절에 따라 다른 액티비티 추천을 받고 싶다. 그래야 사계절 내내 이 서비스를 쓸 수 있다.
- **US-A3** — 초보자로서, 나는 강습 가능 여부·장비 대여·난이도를 미리 알고 싶다. 그래야 처음이라도 부담 없이 도전할 수 있다.
- **US-A4** — 사용자로서, 나는 액티비티에 어울리는 시기(눈·파도·날씨)를 함께 안내받고 싶다. 그래야 헛걸음 없이 제철에 방문할 수 있다.
- **US-A5** — 사용자로서, 나는 액티비티와 주변 휴식(온천·식당)을 한 여정으로 묶어 받고 싶다. 그래야 활동과 회복을 따로 계획하지 않아도 된다.

---

## 추천 시 포함 항목

- 액티비티 종류(스키 / 서핑 / 트레킹 / 자전거 / 패러글라이딩 등)
- 적합 계절·시기(눈·파도·수온·날씨 조건, 월·계절 단위)
- 난이도 · 강습 가능 여부 · 장비 대여 정보
- 추천 이유 + 주변 휴식·식당 코스
- 적합한 동행 유형(친구 / 연인 / 가족)

## 기존 구조 반영 방식

| 반영 위치 | 내용 |
|---|---|
| 테마 가중치 | 기존 6개 → 추가 (`activity`) |
| 소도시 DB | 각 소도시의 `themes` 객체에 `activity` 점수 추가 + 계절별 액티비티 태그 |
| 계절 연계 | **계절축과 직접 곱해지는 핵심 테마** — 시기별 추천 액티비티 분기 |
| 동행 유형 | 친구 / 연인 / 가족 적합 |
| 기존 테마 보완 | 정적·관조형 테마군에 동적·체험형 축을 더해 타겟 확장 |
| 양국 데이터 | 한국(스키·서핑·래프팅) · 일본(스키·서핑·트레킹) 모두 확보 가능 |
