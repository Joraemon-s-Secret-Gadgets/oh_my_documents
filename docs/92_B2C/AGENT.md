# Agent Instructions for 92_B2C

이 폴더는 로브(Lovv)의 B2C 시장 수요, 고객 세그먼트, 과금 가설과 수익성 검증 계획을 관리한다.

## Editing Rules

- 대표 문서는 `92_B2C.md`이다.
- 여행 의향, 검색량, 실제 방문, 숙박, 결제를 서로 다른 증거로 구분한다.
- 국가 전체·광역권·도시·소도시의 범위를 명확히 하고, 관문 대도시를 소도시 수요로 계산하지 않는다.
- 공식 통계와 민간 설문, 언론의 재인용, 내부 가정을 구분하고 조사 기준일과 출처를 명시한다.
- 입국자 수는 고유 이용자 수가 아니므로 여행 빈도나 시장 침투율로 곧바로 해석하지 않는다.
- 과금률, 객단가, CAC, 전환율은 확정값이 아닌 시나리오로 표시하고 산식을 함께 제공한다.
- 내용 변경은 Markdown 원본에 먼저 반영하고, 공유용 HTML은 생성물로 취급한다.
