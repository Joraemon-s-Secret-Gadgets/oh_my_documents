# 로브 프로젝트 색상맵

본 문서는 로브(Lovv) 웹 문서와 향후 화면 구현에 적용할 프로젝트 공통 색상 기준을 정의한다.

## 색상 팔레트

| 이름 | HEX | 용도 |
| --- | --- | --- |
| Main Orange | `#F26518` | 구글 로그인 버튼, 주요 강조 텍스트("지금은 이곳"), 로고에 사용된 활기찬 오렌지 컬러 |
| Dark Charcoal | `#222222` | 타이틀 및 본문 텍스트에 사용된 가독성 높은 짙은 차콜/블랙 컬러 |
| Soft Beige | `#FFF3E7` | 웹페이지의 전체적인 배경을 이루는 부드럽고 따뜻한 연베이지 컬러 |
| Card Peach | `#FCE7DB` | 하단 카드 섹션(숨겨진 장소, 취향 큐레이션)의 배경에 사용된 은은한 피치 베이지 컬러 |
| Muted Brown | `#D44A14` | 상단 소제목("소도시 여행의 새로운 기준") 등에 사용된 차분한 톤다운 오렌지/브라운 컬러 |
| Sub Text Gray | `#666666` | 하단 이용약관이나 본문 설명글에 사용된 부드러운 그레이 컬러 |

## CSS 변수 적용안

```css
:root {
  --main-orange: #F26518;
  --dark-charcoal: #222222;
  --soft-beige: #FFF3E7;
  --card-peach: #FCE7DB;
  --muted-brown: #D44A14;
  --sub-text-gray: #666666;
}
```

## 적용 원칙

- 페이지 전체 배경은 `Soft Beige`를 기본으로 사용한다.
- 본문과 주요 타이틀은 `Dark Charcoal`을 기본 텍스트 컬러로 사용한다.
- 핵심 CTA, 로고, 강한 강조 텍스트는 `Main Orange`를 사용한다.
- 보조 강조 문구와 섹션 소제목은 `Muted Brown`을 사용한다.
- 카드형 정보 영역은 `Card Peach`를 사용하되, 텍스트 대비가 부족하지 않도록 `Dark Charcoal`과 함께 사용한다.
- 약관, 보조 설명, 캡션은 `Sub Text Gray`를 사용한다.
- HTML 생성물에 적용할 때는 원본 Markdown/기획 문서와 충돌하지 않도록 공통 CSS 변수부터 갱신한다.
